CASE

Overview

CASE is used to return specific values when one or more conditions are met. It is used jointly with WHEN, THEN and ELSE clauses.

If none of the WHEN conditions is met, then the result in the ELSE clause will be returned.

If none of the WHEN conditions is met, AND if the ELSE clause is omitted, then NULL will be returned.

Syntax

CASE
    WHEN condition-1 THEN result-1
    WHEN condition-2 THEN result-2
    ...
    WHEN condition-N THEN result-N
    [ELSE result-else]
END [column_name]

Where: condition-n is an expression that has a boolean result result-n is the result to be returned should condition-n be met result-else (optional) is the result to return should none of explicit conditions under WHEN be met. column_name (optional) will be the name of the table
