DROP TABLE

DROP TABLE

DROP TABLE is used to permanently delete a table and its contents.

Syntax

DROP TABLE uses the following syntax

DROP TABLE 'TABLE_NAME';

Examples

DROP TABLE ratings;

When you would like to delete the data inside a table but to keep the table, use TRUNCATE.
