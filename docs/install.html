Installing QuestDB

Supported Platforms

We support the following platforms:

  • Mac OS X 64-bit
  • Windows 64-bit
  • Linux 64-bit

We support JAVA 8 SDK and JRE

32-bit versions are not supported.

We recommend to have Java installed and JAVA_HOME environment variable setup. Both Windows and Linux wrappers for QuestDB will require JAVA_HOME. OSX wrapper will figure out Java location automatically.

Downloading JAVA

QuestDB requires JAVA You can download it here . If you are installing on ARM, download the ARM versions here

Downloading QuestDB

You can download QuestDB binaries and launch scripts here

Installing

Simply extract the files in a directory of your choice. Then to run:

  • On Windows, launch questdb.exe
  • On Linux & MacOS, launch questdb.sh
