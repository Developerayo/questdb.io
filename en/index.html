What is QuestDB?

QuestDB is a high-performance, low-latency time-series database, which runs natively across multiple platforms and architectures. Our approach is a different take on high performance databases, and was born from low latency trading principles.

We achieve maximum efficiency for a given unit of hardware with cutting-edge proprietary software. Core performance has been pushed to the maximum; QuestDB on a single thread is already many times faster than multi-threaded databases.

We provide full stack solutions ready for enterprise, IoT/edge and low-latency trading.

Nanosecond Latency

Unrivaled read / write speed and reliable performance

Full vertical scalability

Run natively on any platform, suited to all kind of specs

We are SQL

We support standard ANSI SQL, with additional augmented features

We are all about performance...

QuestDB is the fruit of 5 years of R&D in high-frequency and low-latency trading environments. Our stack is built from the ground up to maximise performance, efficiency, and reliability.

... and ease of use

Switch from Postgres using the same integrations, which you have already built. Integrate easily with your applications. We support Java natively and can run as a library. We also offer a REST API and web server. Finally, we are SQL.
