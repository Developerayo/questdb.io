Fast relational time-series

QuestDB is a fast NewSQL database for Hybrid Transactional, Analytical and Time Series processing workloads. Users interact via HTTP endpoints, web console, PostgreSQL wire, Influx wire, and programmatic API. The whole database fits in a dependency-free 3.5mb package and is Open Source under Apache 2.0.
